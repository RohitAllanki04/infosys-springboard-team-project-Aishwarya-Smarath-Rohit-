# SmartShelfX Frontend

React-based frontend for SmartShelfX inventory management system.

## Prerequisites

- Node.js 16+ and npm

## Installation

```bash
npm install
```

## Development

```bash
npm start
# or with Vite
npm run dev
```

## Build for Production

```bash
npm run build
```

## Environment Variables

Create a `.env` file:

```
REACT_APP_API_URL=http://localhost:8080/api
```

## Features

- User Authentication (JWT)
- Product Management (CRUD)
- CSV Import
- Stock Transactions
- AI-Based Forecasting
- Purchase Orders
- Alerts & Notifications
- Analytics Dashboard

## Tech Stack

- React 18
- React Router v6
- Axios
- TailwindCSS
- Chart.js
- Lucide React Icons
